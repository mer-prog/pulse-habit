FEATURES

主な機能

⚡
オフラインファースト
SQLiteにローカル保存 → キュー経由でSupabaseに同期。ネット不要で完全動作。
☁️
クラウド同期
Supabase + Row Level Securityでユーザーごとにデータ分離。リトライロジック付き。
🌗
ダークモード
Light / Dark / System。Zustand + useColorSchemeでシームレス切り替え。
🔥
ストリーク追跡
現在のストリーク、最長記録、達成率をリアルタイム計算。SVGリング表示。
🎨
Neo-Brutalist UI
10個のカスタムコンポーネント。太ボーダー、オフセットシャドウ、モノスペースフォント。
🔒
Supabase Auth
メール認証。セッション管理、自動リフレッシュ、RLSによるデータ保護。

SYNC PIPELINE

オフライン同期の仕組み

01
ローカル書き込み
全ての変更はまずSQLiteに書き込み。UIは即座にレスポンス。
02
キューイング
変更をsync_queueテーブルにエンキュー。テーブル名・操作・データを記録。
03
セッション検証
Supabaseセッションの有効性を確認。user_idの不一致を自動修正（ローカルID → auth UID）。
04
RLS通過
Row Level Securityが各行のuser_idとauth.uid()を照合。一致しないデータは拒否。
05
リトライ & パージ
失敗したアイテムはリトライカウンタで再試行。一定回数超過で自動パージ。

毎日の習慣を、一つずつ積み上げよう。

PulseHabitは、Neo-Brutalistデザインの習慣トラッカー。オフラインファーストで動作し、Supabaseでクラウド同期。
  • カスタムアイコン・カラーで無制限に習慣を作成
  • ワンタップで毎日の達成を記録
  • ストリーク表示でモチベーション維持
  • 週間・月間の美しい統計グラフ
  • ダークモード対応（ライト / ダーク / システム）
  • クラウド同期でデバイス間データ共有
  • オフラインファースト — ネット不要で動作
